# minechat-uniapp

## 说明

- 启动页为 `src/pages/webview/webview.vue`，地址：`https://front-dev.agatha.org.cn/`

## Project setup
```
npm install
```

## 运行（开发）

### H5（浏览器）
```
npm run serve
```

等价命令：

```
npm run dev:h5
```

### App（app-plus）

```
npm run dev:app-plus
```

## 构建（生产）

### H5
```
npm run build
```

等价命令：

```
npm run build:h5
```

### App（app-plus）

```
npm run build:app-plus
```
